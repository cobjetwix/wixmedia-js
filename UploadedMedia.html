Class: UploadedMedia

UploadedMedia

new UploadedMedia()

Information about media that was uploaded

Properties:
Name Type Description
fileId string

The id of the uploaded file
imageId string

The id of the uploaded image, if an image
fileUrl string

The url of the uploaded file. This is the same as the fileId
fileSize string

The size of the uploaded file
fileName string

The name of the uploaded file
originalFileName string

The original file name of the uploaded file
width string

If an image, the width of the image
height string

If an image, the height of the image
