Class: WixImage

WixImage

new WixImage(baseUrl, imageId)

a WixImage is a configurable object that supports all the operations, filters and adjustments supported by Wix Media Services

Parameters:
Name Type Description
baseUrl String

the base URL where the image is hosted
imageId String

the id of the image to manipulate
